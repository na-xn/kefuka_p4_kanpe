# 絶妖星乱舞 P4 真偽判定カンペ

FF14「絶妖星乱舞 P4」用の真偽判定カンペアプリです。ゲーム画面の最前面に表示され、クリックしてもゲームの操作（フォーカス）を奪いません。判定を入力すると、自分のやるべき行動が処理順（タイムライン）で表示されます。音声読み上げや、戦闘中に手を離さず判定を入れられるキー入力にも対応しています。

## 起動方法

`p4-kanpe.exe` をダブルクリックするだけで起動します。インストールは不要です。

- Windows 10 / 11 対応
- 初回起動時に「WebView2 ランタイム」が必要です。Windows 11 は標準で入っています。Windows 10 で起動しない場合は、Microsoft の公式ページから「Microsoft Edge WebView2 ランタイム（Evergreen）」を入れてください。
- 発行元不明の警告が出た場合は「詳細情報」→「実行」で起動できます。

## 使い方（2ステップ）

### ① 判定入力（1イベントずつ）
上から順に GC1 → つなみ/ほのお1 → GC2 → つなみ/ほのお2 → GC3 を入力します。各イベントの**見出しの右にある「真／偽」ボタン**でそのフェーズの真偽を選び、本体で自分の担当を選んで「**確定 →**」を押すと次へ進みます。

- **グランドクロス**: 担当を「**雷 / 水 / 加速度**」から選択。担当が決まると「**処理（早／遅）**」が出ます（雷水は処理タイミング、加速度は爆弾の早／遅）。GC2 は GC1 と排他なので自動で選択肢が切り替わります。担当が「加速度」のときだけ「**呪詛（発生源）有／無**」が出ます。
- **つなみ / ほのお**: 「炎 / 水」と「処理（早／遅）」を選択（2回目は1回目と排他で自動）。
- **GC3**: 「アラガン / 死の超越」を選択。
- すべて入力すると最後に処理フローへ進みます。「**← 判定を編集**」でいつでも入力に戻れます。

### ② 処理フロー（タイムライン）
確定後、処理順に自分の行動が並びます。**自分が担当する加速度爆弾・呪詛の発生源のステップは★で強調**されます。途中の判定（無の氾濫・もりもりサンダガ／ひろげるブリザガ・マジックアウト）はこの画面で入力します。

## 音声読み上げ（任意）

右クリックメニューの「**読み上げ**」を ON にすると、処理画面に入った瞬間を 0:00 として、各処理の少し前に行動を読み上げます。

- **開始ホットキー**（既定 `Ctrl+Shift+F4`）でも、その瞬間を 0:00 として読み上げを開始できます。
- ⚙設定で**音量**・各処理の**秒数**を調整できます。
- 「**マジックアウトで踏む/踏まないを読み上げる**」を ON にすると、マジックアウト時にサンダガ／ブリザガの踏む踏まないも読み上げます（既定 OFF）。
- 着弾し終わったステップは自動で淡色化し、最後まで進むと自動で ALL リセットします。
- うまく読まれないときは「**読み上げログを表示**」で各発話の受信状況を確認できます。

## キー入力（任意・F1〜F3）

右クリックメニューの「**キー入力（F1〜F3）**」を ON にすると、**いま入力すべき欄が枠で強調**され、キーで選べます。戦闘中にマウスへ持ち替えずに真偽などを入力できます。

- **F1 / F2 / F3 = 強調枠の 1 / 2 / 3 番目の選択肢**（真偽=真/偽、担当=雷/水/加速度、マジックアウト=雷/氷/両方 など）。入力すると次の欄へ自動で進みます。
- 既定キーは `Ctrl+Shift+F1 / F2 / F3`。⚙設定の「キー入力」で**ドロップダウンから変更**できます（ゲームのホットバーと衝突しないキー推奨）。
- キーは**フォーカスが無くても効く**グローバルショートカットです。押すと画面上部に「🎹 受信」表示が出ます。ON の間、選んだキーはゲーム側へは渡りません。

## 便利機能（右クリックメニュー）

- **位置移動**: 上部バーをドラッグ。**位置ロック**: 上部の鍵アイコンで固定。
- **透過度**スライダー / **自動確定**（全入力後 N 秒で次へ）。
- **①生者の傷・⑧アルテマを隠す**: 不要なステップを省略。
- **ALLリセット**: すべての入力を消去。**閉じる**: 右上の ×。
