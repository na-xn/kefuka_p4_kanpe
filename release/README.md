# 絶妖星乱舞 P4 真偽判定カンペ

FF14「絶妖星乱舞 P4」用の真偽判定カンペアプリです。ゲーム画面の最前面に表示され、クリックしてもゲームの操作（フォーカス）を奪いません。判定を入力すると、自分のやるべき行動が処理順（タイムライン）で表示されます。

## 起動方法

`p4-kanpe.exe` をダブルクリックするだけで起動します。インストールは不要です。

- Windows 10 / 11 対応
- 初回起動時に「WebView2 ランタイム」が必要です。Windows 11 は標準で入っています。Windows 10 で起動しない場合は、Microsoft の公式ページから「Microsoft Edge WebView2 ランタイム（Evergreen）」を入れてください。
- 発行元不明の警告が出た場合は「詳細情報」→「実行」で起動できます。

## 使い方（2ステップ）

### ① 判定入力（1イベントずつ）
上から順に GC1 → つなみ/ほのお1 → GC2 → つなみ/ほのお2 → GC3 を入力します。各イベントの**見出しの右にある「真／偽」ボタン**でそのフェーズの真偽を選び、本体で自分の担当を選んで「**確定 →**」を押すと次へ進みます。

- **グランドクロス**: 担当を「**雷 / 水 / 加早 / 加遅**」から選択（加早=雷水なしで加速度・早、加遅=同・遅）。GC2 は GC1 と排他なので自動で選択肢が切り替わります。担当が「加早/加遅」のときだけ「**呪詛（発生源）有／無**」が出ます。
- **つなみ / ほのお**: 「炎 / 水」を選択。
- **GC3**: 「アラガン / 死の超越」を選択。
- すべて入力すると最後に処理フローへ進みます。「**← 判定を編集**」でいつでも入力に戻れます。

### ② 処理フロー（タイムライン）
確定後、処理順に自分の行動が並びます。**自分が担当する加速度爆弾・呪詛の発生源のステップは★で強調**されます。途中の判定（無の氾濫・もりもりサンダガ／ひろげるブリザガ・マジックアウト）はこの画面で入力します。

## 便利機能

- **位置移動**: 上部バーをドラッグ。
- **位置ロック**: 上部の鍵アイコンで固定（誤操作防止）。
- **透過度 / 自動確定**: ウィンドウ上で**右クリック**。透過度スライダーと、「**自動確定**（全入力後 N 秒で次へ）」の設定があります（戦闘中のクリック削減用）。
- **ALLリセット**: すべての入力を消去。
- **閉じる**: 右上の ×。

## 注意

- ゲームの利用規約の範囲内でご利用ください。
- うまく起動しない・表示が崩れる場合は WebView2 ランタイムの導入をお試しください。
