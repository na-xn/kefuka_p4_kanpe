# 絶妖星乱舞 P4 真偽判定カンペ

FF14「絶妖星乱舞 P4」用の真偽判定カンペアプリです。ゲーム画面の最前面に表示され、クリックしてもゲームの操作（フォーカス）を奪いません。

## 起動方法

`counter-topmost.exe` をダブルクリックするだけで起動します。インストールは不要です。

- Windows 10 / 11 対応
- 初回起動時に「WebView2 ランタイム」が必要です。Windows 11 は標準で入っています。Windows 10 で起動しない場合は、Microsoft の公式ページから「Microsoft Edge WebView2 ランタイム（Evergreen）」を入れてください。
- 発行元不明の警告が出た場合は「詳細情報」→「実行」で起動できます。

## 使い方

- 各ギミックの **「真」「偽」** ボタンを押すと、その場で取るべき行動が下に表示されます。
- 同じボタンをもう一度押すと選択を解除できます。
- **ALLリセット**ボタンで全部の選択をまとめてクリアします。
- ウィンドウは常にゲームの手前に表示され、ボタンを押しても**ゲームはそのまま操作できます**。

## ウィンドウ操作

- **移動**: 上部のバー（タイトル部分）をドラッグ。
- **位置ロック**: 上部の鍵アイコンを押すと固定され、誤って動かさなくなります。もう一度押すと解除。
- **透過度**: ウィンドウ上で**右クリック**するとスライダーが出ます。薄くすると背景のゲームが透けて見えます。
- **閉じる**: 右上の × ボタン。

## 注意

- ゲームの利用規約の範囲内でご利用ください。
- うまく起動しない・表示が崩れる場合は WebView2 ランタイムの導入をお試しください。
